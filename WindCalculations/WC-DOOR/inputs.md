# WC-DOOR Inputs

## Purpose
Site-specific wind pressure certification for exterior door installation.

## Required Inputs
- Property address
- Jurisdiction
- Florida Product Approval (FL#)
- Door size (width x height)
- Installation type (direct to concrete, wood buck, etc.)
- Mean roof height (or documented assumption)

## Required Input Documents
- Florida Product Approval (FL#)
- Manufacturer installation instructions
- Door cut sheet
- Optional elevation drawing
