# WC-ROOF-TILE Inputs

## Purpose
Full reroof wind compliance package for HVHZ tile systems.

## Required Inputs
- Address
- Contractor information
- Roof slope
- Mean roof height
- Roof area (QuickMeasure preferred)
- Tile NOA
- Underlayment NOA
- Fastening method (nails/foam)
- Permit number (if issued)

## Required Input Documents
- Tile NOA (full system)
- Underlayment NOA
- Roof report (QuickMeasure)
- HVHZ form template
