# WC-WINDOW Inputs

## Required Inputs
- Property address
- Jurisdiction
- Florida Product Approval (FL#)
- Window size and configuration
- Mullion condition
- Single unit or combination assembly
- Glazing type (laminated, impact, etc.)
- Installation/substrate condition
- Mean roof height (or documented assumption)

## Required Input Documents
- Florida Product Approval (FL#)
- Manufacturer installation instructions
- Window cut sheets
- Elevation/shop drawing (recommended)
